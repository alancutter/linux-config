In order to use AtomicSkunker, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "AtomicSkunker" );

Then you can create an instance of the plugin object when you need it like this:

 plg = app.CreateAtomicSkunker();

Examples:

Example - Get Version

app.LoadPlugin( "AtomicSkunker" );

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  btn = app.CreateButton( "Press Me" );
  btn.SetOnTouch( CallPlugin );
  lay.AddChild( btn );

  plg = app.CreateAtomicSkunker();

  app.AddLayout( lay );
}

function CallPlugin()
{
  alert( plg.GetVersion() );
}

Example - Create Button

app.LoadPlugin( "AtomicSkunker" );

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" );

  plg = app.CreateAtomicSkunker();

  lay.AddChild( btn );

  app.AddLayout( lay );
